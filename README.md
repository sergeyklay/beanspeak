# Beanspeak

[![Build Status][:badge-travis:]][:build-travis:]

A PHP client library for the [beanstalkd][:beanstalkd:] queue server.

Supported PHP versions: 7.0, 7.1, 7.2, 7.2, 7.3

**Pre-Alpha version, currently not working.**
